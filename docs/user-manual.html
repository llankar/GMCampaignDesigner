Getting Started

  • Launch the app: python main_window.py.
  • Choose or create a campaign database via the left sidebar (Change Data Storage).
  • Populate PCs, NPCs, Creatures, Places, Objects, Informations, Clues, and Maps.

UI Overview

The left sidebar provides quick access to tools and entity managers. Hover each icon to see its name. Click to open the corresponding manager in the main area. Use the top-right banner toggle (when available) to show PCs at a glance.

Managing Entities

Each list supports filtering, sorting by column, and context actions:

  • Add/Edit: Double-click a row or use toolbar buttons to edit details.
  • Duplicate/Delete: Right-click a row to duplicate or delete.
  • Import/Export: Buttons allow saving to and loading from JSON.
  • Columns & Fields: Right-click to show/hide columns and pick which fields display.
  • Colors & Portraits: Assign row colors; show portrait if available.

Right-click menu includes:

  • Open in GM Screen
  • Display on Second Screen
  • Show Portrait
  • Copy
  • Paste
  • Delete
  • Clear Color
  • Columns...
  • Display Fields...
  • Row Color

Clues board: Right-click a card for: Edit Clue, Delete Clue. Right-click a link for: Add Link, Delete Link.

Graph Editors

Visual editors for NPCs, PCs, Factions, and Scenarios let you:

  • Add nodes: Use dedicated add actions or double-click (where available).
  • Drag to move: Left-click and drag nodes to arrange.
  • Create links: Select first node, then second; enter link text when prompted.
  • Zoom/Pan: Mouse wheel to zoom; Shift+wheel to pan horizontally.

Right-click a node for:

  • Delete Node
  • Change Color
  • Display Portrait Window
  • Display Portrait

Right-click a link for:

  • Arrow Mode submenu: No Arrows, Arrow at Start, Arrow at End, Arrows at Both Ends, Arrow Mode

Right-click a shape for:

  • Delete Shape

GM Screen

Browse and present a scenario with a condensed GM-focused layout:

  • Pick a scenario from the list, then drill into details.
  • Search: Press Ctrl+F to open global search within GM Screen.
  • Banner: Show PCs at the top for quick reference.

Scenario Tools

Two helpers streamline content:

  • Generator: Creates scenario outlines; adjust options, then generate and refine.
  • Importer: Parse external scenario documents and map fields before saving.

Map Tool

Open a separate window to display and edit maps, tokens, auras, fog, and shapes.

  • Select a Map: Choose from your maps list.
  • Fog of War: Add/Remove/Clear/Reset fog; [ and ] adjust brush size; Rectangle/Circle shapes.
  • Tokens: Add NPC/Creature/PC tokens; adjust Token Size slider.
  • Shapes: Switch Active Tool to Rectangle or Oval; set Fill/Border colors and fill mode.
  • Fullscreen/Web: Mirror the map to a separate window or a web client.
  • Context Menus: Right-click tokens/shapes for actions like resize, color, z-order, copy, delete.

Token right-click includes:

  • Resize Token
  • Change Border Color
  • Copy Token
  • Delete Token
  • Show Portrait

Shape right-click includes:

  • Edit Shape Graphically
  • Copy Shape
  • Delete Shape
